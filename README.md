# 🀄 万能麻将 - 中国麻将大全

一个功能完整、支持中国境内所有麻将种类的网页版麻将游戏。

## ✨ 功能特性

### 🎮 游戏模式
- **人机对战** - 1人 vs 2~3个AI（支持多种难度）
- **局域网联机** - 3~4人 WebRTC 对战（需信令服务）
- **自定义模式** - 自由选择麻将种类与规则
- **牌局回放** - 查看历史对局记录

### 🀅 支持的麻将种类
- 国标麻将（144张）
- 广东麻将（推倒胡）
- 四川麻将（血战到底、缺一门）
- 上海麻将（花牌计番）
- 北京麻将（混儿牌）
- 台湾麻将（16张）
- 杭州麻将（财神牌）
- 长沙麻将（扎鸟）
- 东北麻将（带会儿）
- 湖北麻将（开口翻）
- 福建麻将（金牌）
- 江西麻将（精牌）
- 湖南三人麻将（红中赖子）
- 四川三人麻将

### 🎯 核心功能
- 完整的吃、碰、杠、胡规则
- 自摸、杠上开花、海底捞月
- 七对、十三幺等特殊牌型
- 暗杠、明杠、加杠
- 花牌补牌
- 流局判定
- 番数计算

### 🎨 个性化设置
- 4种牌桌主题（经典绿、深海蓝、原木色、中国红）
- 对手手牌显示方式（隐藏/小牌堆/完整显示）
- 牌桌缩放、手牌大小调整
- 动画强度控制
- BGM/SFX音量控制
- 游戏速度调节

### 📊 统计与成就
- 等级与经验系统
- 胜率、连胜记录
- 总得分、最高单局
- 20+成就待解锁
- 历史对局记录

## 🚀 快速开始

### 本地运行
```bash
# 终端 1：启动静态页面
npm run client

# 终端 2：启动联机信令服务
npm run server
```

然后在浏览器中打开 `http://localhost:8080`

### 一键启动（仅适合 macOS / Linux shell）
```bash
npm start
```

### 直接打开
由于游戏使用纯前端技术（HTML/CSS/JavaScript），也可以直接在浏览器中打开 `index.html` 文件。

**注意**：
- 局域网联机需要同时启动 `8080` 静态页面和 `8081` 信令服务。
- 直接打开 `index.html` 只适合单机查看，不适合联机。

## 🖥️ 浏览器支持
- Chrome / Edge / Safari / Firefox 最新版
- 支持移动端浏览器
- 推荐分辨率：1280×720 及以上

## 🎮 操作说明

### 游戏内操作
- **点击手牌** - 选中/取消选中
- **再次点击已选中的牌** - 打出
- **1** - 吃
- **2** - 碰
- **3** - 杠
- **4 / 空格** - 胡
- **S** - 跳过
- **ESC** - 打开游戏菜单

### 快捷键
- 支持键盘快捷键进行吃碰杠胡操作
- 移动端支持触摸手势
